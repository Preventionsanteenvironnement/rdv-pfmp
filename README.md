# Rendez-vous PFMP

Outil de prise de rendez-vous pour les visites de stage (PFMP).
Le tuteur reçoit un lien unique, choisit ses créneaux par préférence, et reçoit une carte de confirmation.

Isolé du portail Coordination & Pédagogie. Données : Firestore `devoirs-pse` / collection `coordination_rdv`.

⚠️ Aucune donnée nominative : initiales élève uniquement, jamais le nom de l'établissement.
